DRUCKA — PHOTO MOCKUPS FOLDER
=============================

The editor loads blank-product photos from here. Missing files show a
"Mockup image missing" card with the exact expected filename.
Each view tries: selected colour (.png, .jpg) → white (.png, .jpg).

NAMING (all lowercase):
  adult flat:   tshirt-front-white.png       tshirt-back-white.png
  adult male:   tshirt-male-front-white.png  tshirt-male-back-white.png
  adult female: tshirt-female-front-white.png tshirt-female-back-white.png
  kids flat:    kids-tshirt-front-white.png  kids-tshirt-back-white.png
  kids model:   kids-model-front-white.png   kids-model-back-white.png

  Replace "white" with: black | navy | red | yellow
  Same pattern for: oversized, hoodie, kids-hoodie, mug, frame,
  cushion, canvas, keychain (e.g. hoodie-front-black.png)

PHOTO SPECS:
  - Portrait 42:50 ratio (e.g. 840 x 1000 px), product centered, blank
  - Plain light background, under ~300 KB (JPG) / ~800 KB (PNG)

PRINT-AREA ALIGNMENT:
  Tune per product/view/side in src/App.jsx -> PHOTO_AREAS, or live with
  the "Adjust print area" gear panel in the editor.
